## 🎯 主な機能

1. **クリックして推論**: トレーニング済みモデルカードをクリックして推論ダイアログを開く
2. **スマートデフォルト**: 各データセットに最適化されたデフォルトパラメータ
3. **サンプルプロンプト**: 各データセットタイプ向けのクイックスタートテンプレート
4. **リアルタイム生成**: オンデマンドでテキストサンプルを生成
5. **複数サンプル**: 一度に 1〜10 サンプルを生成
6. **細かな制御**: 温度、top-k、最大長を調整可能
7. **結果のコピー**: ワンクリックでクリップボードにコピー
8. **ビジュアルフィードバック**: ローディング状態とエラーを含むプロフェッショナルな UI

## 📊 対応データセット

- ✅ Genome Sequence（DNA）
- ✅ Protein Sequence（アミノ酸）

## 🚀 使用方法

1. **サーバーを起動**:

   ```bash
   cd molcrawl-web
   export LEARNING_SOURCE_DIR="learning_source_202508"
   npm start
   ```

2. **GPT-2 トレーニングステータスページに移動**

3. **トレーニング済みモデルカードをクリック**（チェックポイントデータがあるカード）

4. **モーダルが開きます**:
   - モデル情報
   - 生成パラメータ
   - サンプルプロンプト

5. **パラメータを調整**して「Generate」をクリック

6. **結果を確認してコピー**

- PyTorch と Transformers を含む Python 環境が必要です
- 高速推論には GPU が推奨されます

- すべてのチェックポイント形式を自動サポートします
- フロントエンドへの追加の依存関係は不要です
