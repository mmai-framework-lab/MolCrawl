## コンポーネントアーキテクチャ

```
┌──────────────────────────────────────────────────────────────────────┐
│                            ブラウザ                                   │
│                                                                       │
│  ┌────────────────────────────────────────────────────────────────┐ │
│  │        GPT2TrainingStatus.js（メインコンポーネント）           │ │
│  │                                                                 │ │
│  │  State:                                                         │ │
│  │  • trainingData      → モデルステータスとメトリクス            │ │
│  │  • modalOpen         → モーダルの表示状態                      │ │
│  │  • selectedModel     → 現在選択されているモデル                │ │
│  │                                                                 │ │
│  │  ┌──────────────────────────────────────────────────────────┐ │ │
│  │  │  モデルカード（グリッドレイアウト）                      │ │ │
│  │  │                                                           │ │ │
│  │  │  [Small]  [Medium]  [Large]  [XL]                        │ │ │
│  │  │     ↓        ↓         ↓       ↓                          │ │ │
│  │  │  onClick  onClick  onClick  onClick                       │ │ │
│  │  │     ↓        ↓         ↓       ↓                          │ │ │
│  │  │  handleModelClick() → setModalOpen(true)                 │ │ │
│  │  └──────────────────────────────────────────────────────────┘ │ │
│  │                          ↓                                      │ │
│  │  ┌──────────────────────────────────────────────────────────┐ │ │
│  │  │     InferenceModal.js（モーダルコンポーネント）          │ │ │
│  │  │                                                           │ │ │
│  │  │  Props:                                                   │ │ │
│  │  │  • isOpen            → boolean                           │ │ │
│  │  │  • dataset           → "compounds", "rna" など           │ │ │
│  │  │  • size              → "small", "medium" など            │ │ │
│  │  │  • modelData         → チェックポイント情報              │ │ │
│  │  │                                                           │ │ │
│  │  │  State:                                                   │ │ │
│  │  │  • prompt            → 入力テキスト                      │ │ │
│  │  │  • maxLength         → 生成長                            │ │ │
│  │  │  • temperature       → サンプリング温度                  │ │ │
│  │  │  • topK              → top-k サンプリング                │ │ │
│  │  │  • results           → 生成テキスト                      │ │ │
│  │  │                                                           │ │ │
│  │  │  ┌────────────────────────────────────────────────────┐ │ │ │
│  │  │  │  [生成ボタン]                                      │ │ │ │
│  │  │  │       ↓                                            │ │ │ │
│  │  │  │  handleInference()                                 │ │ │ │
│  │  │  │       ↓                                            │ │ │ │
│  │  │  │  POST /api/gpt2-inference                          │ │ │ │
│  │  │  └────────────────────────────────────────────────────┘ │ │ │
│  │  └──────────────────────────────────────────────────────────┘ │ │
│  └────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────┘
                              ↓ HTTP POST
┌──────────────────────────────────────────────────────────────────────┐
│                         Express サーバー                              │
│                                                                       │
│  server.js                                                           │
│  ├── ルート: /api/gpt2-inference                                     │
│  │      ↓                                                            │
│  └── gpt2-inference.js                                               │
│       │                                                              │
│       ├─→ validateParams()                                           │
│       ├─→ getCheckpointPath(dataset, size)                           │
│       │    ├─→ HuggingFace 形式を確認                                │
│       │    └─→ レガシー形式を確認                                    │
│       │                                                              │
│       └─→ runInference()                                             │
│            │                                                          │
│            ├─→ Python スクリプトを生成                               │
│            ├─→ spawn('python', ['-c', script])                       │
│            │                                                          │
│            ↓                                                          │
└──────────────────────────────────────────────────────────────────────┘
                              ↓ Python プロセス
┌──────────────────────────────────────────────────────────────────────┐
│                         Python 推論                                   │
│                                                                       │
│  1. ライブラリのインポート（torch, transformers）                    │
│  2. モデルの読み込み:                                                │
│     ├─→ HuggingFace: GPT2LMHeadModel.from_pretrained()              │
│     └─→ レガシー: torch.load(ckpt.pt)                               │
│  3. トークナイザーの読み込み                                         │
│  4. プロンプトのエンコード                                           │
│  5. 生成:                                                            │
│     model.generate(                                                   │
│       input_ids,                                                      │
│       max_length=maxLength,                                          │
│       temperature=temperature,                                        │
│       top_k=topK                                                      │
│     )                                                                 │
│  6. 結果のデコード                                                   │
│  7. JSON を stdout に出力                                            │
│                                                                       │
│  出力: {"success": true, "results": ["text1", "text2"]}             │
└──────────────────────────────────────────────────────────────────────┘
                              ↓ JSON レスポンス
┌──────────────────────────────────────────────────────────────────────┐
│                        ブラウザに戻る                                 │
│                                                                       │
│  InferenceModal.js                                                   │
│  ├── レスポンスを受信                                                │
│  ├── setResults(response.results)                                    │
│  ├── 結果を表示                                                      │
│  └── コピーボタンを有効化                                            │
└──────────────────────────────────────────────────────────────────────┘
```

## API リクエスト/レスポンスの流れ

```
フロントエンド                バックエンド                 Python
   │                           │                            │
   │  POST /api/gpt2-inference │                            │
   │  {                        │                            │
   │    dataset: "compounds",  │                            │
   │    size: "small",         │                            │
   │    prompt: "<|start|>C",  │                            │
   │    maxLength: 128,        │                            │
   │    temperature: 1.0       │                            │
   │  }                        │                            │
   ├──────────────────────────►│                            │
   │                           │  パラメータを検証          │
   │                           │  チェックポイントを検索    │
   │                           │  Python スクリプトを生成   │
   │                           ├───────────────────────────►│
   │                           │                            │ モデルを読み込む
   │                           │                            │ トークナイザーを読み込む
   │                           │                            │ プロンプトをエンコード
   │                           │                            │ テキストを生成
   │                           │◄───────────────────────────┤
   │                           │  JSON: {success, results}  │
   │◄──────────────────────────┤                            │
   │  レスポンス: {            │                            │
   │    success: true,         │                            │
   │    results: [             │                            │
   │      "CC(C)C...",         │                            │
   │      "c1ccccc1..."        │                            │
   │    ]                      │                            │
   │  }                        │                            │
   │  結果を表示               │                            │
   └───────────────────────────┴────────────────────────────┘
```

## エラーハンドリングの流れ

```
試行: API リクエスト
 ├─→ 成功
 │   ├─→ JSON をパース
 │   ├─→ 結果を表示
 │   └─→ コピーボタンを有効化
 │
 └─→ 失敗
     ├─→ ネットワークエラー
     │   └─→ 表示: "Network request failed"
     ├─→ モデルが見つからない
     │   └─→ 表示: "No checkpoint found"
     ├─→ Python エラー
     │   └─→ 表示: stderr の出力
     └─→ 無効なパラメータ
         └─→ 表示: "Invalid parameters"
```
