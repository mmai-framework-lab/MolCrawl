MolCrawl

Foundation Models for Molecules, Proteins, RNA, and Genomic Sequences

Get Started API Documentation

Features

Compound Processing

SMILES tokenization, ChEMBL dataset preparation, and molecular language modeling with GPT-2.

Protein Sequences

ESM-2 fine-tuning, protein sequence datasets, and evaluation pipelines.

RNA Analysis

RNA secondary structure prediction with RNAformer and sequence encoding.

Genomic Sequences

DNABERT-2 integration for genomic sequence understanding and analysis.

Supported Models

GPT-2

For molecular SMILES generation and compound language modeling.

ChemBERTa-2

Pre-trained transformer for molecular property prediction.

ESM-2

State-of-the-art protein language model from Meta AI.

DNABERT-2

Foundation model for multi-species genome understanding.

RNAformer

Transformer architecture for RNA structure prediction.

BERT

General-purpose BERT for clinical variant analysis (ClinVar).

Getting Started

# Clone the repository
git clone https://github.com/mmai-framework-lab/MolCrawl.git
cd MolCrawl

# Create conda environment
conda env create -f environment.yaml
conda activate molcrawl

# Install the package
pip install -e .

For detailed documentation, visit the API Documentation.
